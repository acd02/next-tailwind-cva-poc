# Next.js Starter Kit (with typescript)

## Usage:

- Clone the repo
- install the dependencies `$ npm install`
- dev mode `$ npm run dev`
  - if getting and error, run `$ npm run build`, then `$ npm run dev`
- build `$ npm run build`
- test `$ npm run test`

## Features:

- Styles: [Tailwind](https://tailwindcss.com/)
